# Vishwasānti - विश्वासांति
## The Peace That Comes From Trust

### Our Dharma:
To create a sanctuary where technology meets tranquility, 
building trust that leads to genuine inner peace.

### Core Principles:
1. **Vishwas** (Trust) - Every interaction builds confidence
2. **Shanti** (Peace) - Every feature creates calm  
3. **Sambandh** (Connection) - Every connection heals
4. **Gyaan** (Wisdom) - Every insight enlightens

### Tech Stack:
- Frontend: React Native + TypeScript
- Backend: Node.js + FastAPI
- AI: Python + TensorFlow
- Database: PostgreSQL
